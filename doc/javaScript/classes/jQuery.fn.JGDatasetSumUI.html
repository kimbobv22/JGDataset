jQuery.fn.JGDatasetSumUI Class

Defined in: JGDatasetSumUI.2.1.0.js:26

HTML과 JGDataset의 통계매핑을 위한 플러그인입니다.
함수호출 형식은 아래와 같습니다.

//1. jQuery Plugin Style
var result_ = $(target_).JGDatasetSumUI("함수명",...);

//2. normal Style
var datasetSumUI_ = $(target_).JGDatasetSumUI();
var result_ = datasetSumUI_.함수명();

datasetSumUI 샘플은 여기에서 확인할 수 있습니다.
