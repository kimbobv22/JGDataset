JGDataset Class

JGDataset은 행열단위 작업을 보다 편리하게 할 수 있도록 합니다.
아래는 JGDataset의 JSON 포맷입니다.

1. 스키마 포함 JSON 포맷
    {"columninfo":
        [{"name":"열1","isKey":false}
        ,{"name":"열2","isKey":false}]
    ,"rowdata":
        [{"row":{"열1":{"value":"열값"}
            ,"열2":{"value":"열값"}}
        ,"status":0}
        ,{"row":{"열1":{"value":"열값"}
            ,"열2":{"value":"열값"}}
        ,"status":0}
        ,...]
    ,"deletedRowdata":
        [{"row":{"열1":{"value":"열값"}
            ,"열2":{"value":"열값"}}}
        ,...]
    }

2. 스키마 미포함 JSON 포맷
    [{열1 : 열값, 열2 : 열값, ... }
    ,{열1 : 열값, 열2 : 열값, ... }
    ,...]

JGDataset 샘플은 여기에서 확인할 수 있습니다.
