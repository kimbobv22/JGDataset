jQuery.fn.JGDatasetUI Class

Defined in: JGDatasetUI.2.0.2.js:263

HTML과 JGDataset의 매핑을 위한 플러그인입니다.
함수호출 형식은 아래와 같습니다.

//1. jQuery Plugin Style
var result_ = $(target_).JGDatasetUI("함수명",...);

//2. normal Style
var datasetUI_ = $(target_).JGDatasetUI();
var result_ = datasetUI_.함수명();

JGDatasetUI 샘플은 여기에서 확인할 수 있습니다.
